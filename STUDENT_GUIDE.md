# Student Guide...

## What does this repo do?

Provides C++-speed MIDI manipulation for batch processing. When you need to transpose 10,000 MIDI files at once, this is the tool.

## How it connects

**Anvil** handles the heavy lifting. When Rhapsodia generates 100+ MIDI files, Anvil processes them at C++ speed.
